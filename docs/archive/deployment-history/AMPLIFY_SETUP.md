## 🌐 إعداد Custom Domain (app.TadFuq.ai)

### **الخطوة 1: إضافة Domain في Amplify**

```bash
# عبر CLI
aws amplify create-domain-association \
  --app-id d1pdf5yf8mnktj \
  --domain-name TadFuq.ai \
  --sub-domain-settings prefix=app,branchName=main \
  --region me-south-1
```

أو عبر Console:
1. اذهب إلى **Domain management**
2. اضغط **Add domain**
3. أدخل: `TadFuq.ai`
4. Subdomain: `app`

### **الخطوة 2: تحديث DNS في Route53**

Amplify سيعطيك CNAME record. أضفه:

```bash
# عبر Terraform
cd infra/terraform
```

أضف في `route53.tf`:

```hcl
resource "aws_route53_record" "app" {
  count   = var.domain_name != "" ? 1 : 0
  zone_id = local.hosted_zone_id
  name    = "app.TadFuq.ai"
  type    = "CNAME"
  ttl     = 300
  records = ["<amplify-domain>.cloudfront.net"]
}
```

ثم:
```bash
terraform apply
```

### **الخطوة 3: انتظر SSL Certificate**

Amplify سينشئ SSL certificate تلقائياً (5-10 دقائق).

## 🔄 تحديث CORS في Backend

بعد النشر، حدّث backend للسماح بـ frontend domain:

```bash
cd backend/internal/adapter/http
```

في ملف CORS configuration، أضف:

```go
AllowedOrigins: []string{
    "https://app.TadFuq.ai",
    "https://d1pdf5yf8mnktj.amplifyapp.com",
    "http://localhost:3000", // للتطوير
}
```

ثم أعد نشر backend:

```bash
cd backend
docker buildx build --platform linux/amd64 -t cashflow/tenant-service:latest -f ../infra/docker/Dockerfile . --load
docker tag cashflow/tenant-service:latest 747253121951.dkr.ecr.me-south-1.amazonaws.com/cashflow/tenant-service:latest
docker push 747253121951.dkr.ecr.me-south-1.amazonaws.com/cashflow/tenant-service:latest

aws ecs update-service \
  --cluster cashflow-dev-cluster \
  --service cashflow-dev-tenant \
  --force-new-deployment \
  --region me-south-1
```
